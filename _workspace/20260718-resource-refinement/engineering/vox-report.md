## Vox Director contracts applied

### Prompt guide

The prompt pack uses the five-part image structure for every shot: identical collage style block, discrete layered source pieces, one bold paper background, an exact short headline on title shots, and a 16:9/2k technical clause. The custom theme combines dark-fantasy field-manual subject matter with Russian Constructivist diagonals, risograph/photocopy finish, void black, bone cream, sovereign violet, cold-steel blue, and ember-orange signals. Prompts explicitly preserve printed grain, torn/scissor edges, tape, halftone, paper shadows, rigid layers, and supplied asset identity; they explicitly reject a newly invented CGI/3D render.

Motion prompts use one camera move per shot plus rich rigid-paper element movement, a single continuous settle, flat dimensional lock, and stable title/layout anchors. The one-off traveling signal ribbon is limited to beat 3 rather than repeated formulaically.

### Beat layer

The assignment-fixed four-beat `hook_payoff` progression is:

| Beat | Role | Exact EN / KO headline | Shot sizes | Camera moves |
|---:|---|---|---|---|
| 1 | Hook | `ASSETS ≠ ARMIES` / `에셋 ≠ 군대` | `EST_WIDE` → `DETAIL` | `push_in` → `pan` |
| 2 | Context | `CUT INTO SYSTEMS` / `시스템으로 자른다` | `WIDE` → `CLOSE` | `parallax` → `tilt` |
| 3 | Build | `RULES MAKE WAR` / `규칙이 전장을 만든다` | `WIDE` → `CLOSE` | `pull_out` → `element` |
| 4 | Payoff | `THE ABYSS PLAYS` / `심연이 움직인다` | `EST_WIDE` → `WIDE` | `push_in` → `static` |

Each shot is exactly 3 seconds in the beat map. The first title lands in the first 3 seconds. Adjacent moves never repeat (`push_in → pan → parallax → tilt → pull_out → element → push_in → static`), move families alternate, and `static` is reserved for the final payoff.

### Models and gotchas

The public Atlas model list was checked on 2026-07-18 before model IDs were recorded. The live `display_console` list contained:

- `google/nano-banana-2/text-to-image`
- `google/gemini-omni-flash/image-to-video`
- `xai/tts-v1`

`minimax/music-2.6` was not returned by that public list, so the prompt pack marks it unavailable rather than claiming a live model. Because the Atlas key was absent, the final used local macOS speech synthesis and a quiet procedural two-tone drone. No celebrity/brand animation path was required.

The ffmpeg path follows the documented constraints: Pillow renders bilingual text into the posters; all shot clips are normalized before xfade; the output is H.264 with `yuv420p`, 24 fps, AAC audio, and `+faststart`. Frame sampling uses `format=yuvj420p`.

### Local engine / fallback

The local edition uses repository images as source-identified paper cards rather than pretending to be Atlas generations. Pillow constructs irregular torn masks, paper borders, tape, halftone texture, print grain, diagonal signal fields, bilingual banners, and drop shadows. ffmpeg `zoompan` realizes the safe flat-art camera moves; `element` maps to a locked camera so the poster itself carries the emphasis. Seven varied 0.2-second `xfade` transitions join the eight 3-second shots. The result is a reproducible Ken Burns/xfade fallback, not a claim of AI living-poster or background-removal output.
