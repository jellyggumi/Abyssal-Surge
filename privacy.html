Abyssal Command Analytics Disclosure

We use Cloudflare Web Analytics to measure traffic and page-performance on this host (jellyggumi.github.io). This helps us understand how the game loads and behaves across different browsers and regions.

What We Collect

  • No cookies or local storage are used by our analytics.
  • No custom gameplay events or actions inside the game are tracked.
  • No application identifiers or user accounts are created or monitored.
  • Cloudflare processes only standard HTTP request information (such as browser type, referrer, and approximate geographic location derived from IP address) to compile aggregate performance metrics.

Opt-in / Consent

Analytics are blocked by default. Script loading and traffic telemetry only begin if you explicitly click the "Accept" button on our home page consent banner.
