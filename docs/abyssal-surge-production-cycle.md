# Abyssal Surge production cycle

Abyssal Surge is an **original-only** mobile-first RTS–RPG campaign built for static GitHub Pages delivery. The public target is a three-stage single-player campaign: Cinder Span, Veil Citadel, and Echo Throne.

## The first playable beat

Cinder Span asks players to hunt Ash Echoes, extract a time-limited soul pool, materialize a compact shadow legion, capture the Sable Relay, and defeat the Rift Guardian. Completion offers one of two exclusive rewards that carries into Veil Citadel.

## What is local today

Campaign saves are designed for IndexedDB, with export/import recovery and versioned offline assets. GitHub Pages stays the static client host. This cycle does not claim cloud saves, real-time co-op, ranked play, payments, or APK packaging.

## What follows

Veil Citadel carries the first reward forward, introduces possession, and requires holding two nodes before its tactical boss. Echo Throne carries the second reward, uses a bounded Lord’s Domain comeback action, and ends with the Gate Sovereign.

Before each expansion, the team records evidence against narrative consistency, balance, accessibility/readability, performance, fairness, replayability, and novelty gates.
